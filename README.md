# 🎯 새총의 신 (Saeчong The God)

> 날아가는 새를 새총으로 맞히는 원터치 하이퍼 캐주얼 슈팅 게임  
> Phaser.js 3 + Vite + TypeScript | 앱인토스(App In Toss) 출시 목표

---

## 🚀 실행 방법

### 개발 서버 시작
```bash
npm install       # 최초 1회
npm run dev       # http://localhost:5173 에서 확인
```

### 프로덕션 빌드
```bash
npm run build     # dist/ 폴더에 빌드 결과물 생성
npm run preview   # 빌드 결과물 로컬 미리보기
```

---

## 🎮 게임 조작법

| 동작 | 방법 |
|------|------|
| **조준** | 화면 아무 곳이나 터치 후 드래그 |
| **파워 조절** | 드래그 거리 = 발사 파워 (멀리 당길수록 강하게) |
| **방향 설정** | 드래그 방향의 **반대** 방향으로 발사됨 (새총 원리) |
| **발사** | 손가락을 떼면 발사! |
| **궤적 확인** | 드래그 중 점선으로 예상 궤적 실시간 표시 |

## 🐦 새 종류

| 새 | 색상 | 크기 | 속도 | 특징 |
|----|------|------|------|------|
| 🐦 참새 (Sparrow) | 갈색 | 작음 | 빠름 | 가장 흔함 |
| 🦅 독수리 (Eagle) | 진갈색 | 큼 | 느림 | 양 날개 펼침, 붉은 눈 |
| 🕊️ 비둘기 (Pigeon) | 회색 | 중간 | 보통 | 밝은 가슴 |
| 🦜 앵무새 (Parrot) | 초록 | 중간 | 약간 빠름 | 빨간 볼 포인트 |
| 🦉 올빼미 (Owl) | 브라운 | 가장 큼 | 느림 | 노란 큰 눈, 희귀 (5%) |

### 비행 패턴
- **직선 (straight)**: 수평으로 날아감
- **지그재그 (zigzag)**: 위아래로 흔들며 비행
- **하강 (dive)**: 아래로 기울며 날아감
- **가속 (accelerate)**: 점점 빨라짐

## 🏆 레벨 시스템

| 레벨 | 목표 명중 | 새 기본 속도 |
|------|---------|------------|
| 1 | 6마리 | 150 |
| 2 | 15마리 | 180 |
| 3 | 24마리 | 210 |
| N | 15 + (N-2)×9 | 150 + (N-1)×30 |

- 레벨당 **Miss 3회** 허용 → 3회 초과 시 레벨 재시작
- **게임 오버**: 기회 소진 → 광고 시청 or 15분 대기

## 📁 프로젝트 구조

```
src/
├── constants/
│   └── TDS.ts          # Toss Design System 색상/폰트 상수
├── objects/
│   ├── Bird.ts         # 새 오브젝트 (5종 + 4가지 비행 패턴)
│   └── Projectile.ts   # 투사체 (포물선 물리 + 잔상 효과)
├── scenes/
│   ├── MainMenuScene.ts   # 메인 메뉴
│   ├── GameScene.ts       # 게임 씬 (드래그 조준, 궤적 미리보기)
│   ├── GameOverScene.ts   # 게임 오버 (광고 이어하기)
│   └── LevelUpScene.ts    # 레벨업 연출
├── ui/
│   ├── Button.ts       # TDS 스타일 버튼 컴포넌트
│   ├── HUD.ts          # 상단 HUD (레벨/명중수/미스)
│   └── GaugeBar.ts     # 하단 파워 게이지 바
└── utils/
    └── GameManager.ts  # 게임 전역 상태 싱글톤
```
